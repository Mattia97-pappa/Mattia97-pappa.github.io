Library📚

Libraries connected to each other to allow users to borrow books. Users can search for available titles, place a reservation, and have the book delivered to the library closest to their city.

___________________________________________________________________________

Technologies 🖥️

Java, Spring Framework, Spring Security, JPA (Hibernate), Bootstrap, JavaScript, jQuery

___________________________________________________________________________

User experience demonstration👤

The user can browse available books, make reservations, and track their bookings directly from their personal profile page.
