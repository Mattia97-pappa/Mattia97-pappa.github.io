Hotel🏨

This is a full-stack Java-based web application for managing bookings, rooms, and users. The system is designed for admin use only. The admin has full access to all data and can perform complete CRUD operations on bookings, rooms, and guests.

___________________________________________________________________________

Technologies 🖥️

Java, Spring Framework, Spring Security, JPA (Hibernate), Bootstrap, JavaScript, jQuery

___________________________________________________________________________

Admin experience demonstration🛡️

The admin has CRUD (Create, Read, Update, Delete) permissions to manage the application's data and resources.
