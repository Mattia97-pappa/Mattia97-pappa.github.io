Online Clothing Store 👕🛍️

This is a full-stack Java-based e-commerce web application. The system includes user authentication, product browsing, order placement, and persistent data storage using a relational database. It features a secure login system with Spring Security.

___________________________________________________________________________

Technologies 🖥️

Java, Spring Framework, Spring Security, JPA (Hibernate), Bootstrap, JavaScript
