City Tours🌃

This is a full-stack Java-based web application for managing bookings for people who want to participate in tours while traveling. The system allows users to browse available tours, make bookings, and securely manage their personal information. In a relational database.

___________________________________________________________________________

Technologies 🖥️

Java, Spring Framework, Spring Security, JPA (Hibernate), Bootstrap, JavaScript
