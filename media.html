Mean Math Simulator

I used Scanner for user input and a try-catch block to handle input errors (InputMismatchException). This Java program simulates a school grade management system that calculates the mean and the mean absolute deviation of the entered grades. The mean absolute deviation measures how much each grade, on average, differs from the mean. The user enters up to 15 grades. It calculates the average (mean) of the grades. It then calculates the mean absolute deviation, which is the average of the absolute differences between each grade and the mean.
